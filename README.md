# TeamHub

Team management SaaS app with attendance tracking, task management, and daily reports.

## Features
- Admin & Member dashboards
- Check-in with 15-min window
- Late tracking (3 lates = 3% penalty, 9 lates = 1 strike)
- Tasks with priority, category, deadlines
- Daily reports
- Real-time Supabase database

## Tech Stack
- React 18 + Vite
- Supabase (database)

## Deploy
1. Click "Deploy" on Vercel
2. Connect GitHub repo
3. Done!
